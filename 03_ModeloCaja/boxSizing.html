El box model o "modelo de cajas" es el comportamiento del CSS que hace que todos los elementos de la página se representen mediante cajas rectangulares

box-sizing: content-box;
width: 500px;
padding: 0px;
Por default todos los elementos de bloque tendrán la propiedad box-sizing
establecida a content-box

box-sizing: content-box;
width: 500px;
padding: 20px
Se recomienda siempre cambiar la propiedad box-sizing a border-box
esto hará que se incluya el border y el padding dentro de la propiedad width

box-sizing: border-box;
padding: 50px
box-sizing: border-box
